# Java-Vityarthi

A structured collection of Java programs and concepts learned through the Vityarthi Java Programming course, covering fundamentals to advanced Object-Oriented Programming (OOP).

🚀 About the Project

This repository contains well-organized Java programs, practice questions, and concept-based implementations to strengthen core programming skills.

The content is inspired by structured Java learning modules including:

OOP concepts (Encapsulation, Inheritance, Polymorphism, Abstraction)
Constructors and class design
Interfaces and abstract classes
Exception handling and recursion
Design patterns and real-world modeling
🎯 Objectives
Build strong fundamentals in Java
Practice real-world programming problems
Understand object-oriented design
Prepare for coding interviews and academics

🛠️ How to Run
1. Clone the repository
git clone https://github.com/Sampadaseth/Java-Vityarthi.git
2. Navigate to project folder
cd Java-Vityarthi
3. Compile a Java file
javac FileName.java
4. Run the program
java FileName
📌 Requirements
Java JDK 8 or above
Any IDE (IntelliJ, Eclipse, VS Code)
